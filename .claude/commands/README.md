# Vita Agent Team

> 16 specialized AI agents for building Vita efficiently. Each agent is a Claude Code slash command you can invoke anytime.

| Command | Role | When to Use |

| `/project:design-agent` | Design System Lead | Visual design, component styling, design tokens |
| `/project:code-review-agent` | Staff Engineer | Code quality, bug detection, pattern compliance |
| `/project:qa-agent` | QA Lead | Test strategy, test writing, bug hunting |
| `/project:performance-agent` | Performance Engineer | Frame rates, bundle size, rendering optimization |
| `/project:security-agent` | Security Lead | Auth hardening, RLS policies, vulnerability audit |

| `/project:marketing-agent` | Growth Lead | ASO, go-to-market, user acquisition, copy |

## How to Use

### Invoke any agent
```
/project:ux-agent Audit the onboarding flow for friction points
/project:security-agent Review RLS policies for the messages table
/project:pm-agent Plan the next sprint — what should we build first?
```

### Pass context with your request
```
/project:design-agent Review the SwipeDeck component for visual consistency
/project:qa-agent Write tests for the health ring calculation logic
/project:performance-agent Profile the Groups screen — FlatList is janking
```

### Chain agents for comprehensive work
```
1. /project:pm-agent What are our top 3 priorities for launch?
2. /project:ux-agent Audit the #1 priority feature for UX issues
3. /project:design-agent Fix the visual issues found by the UX agent
4. /project:qa-agent Write tests for the changes
5. /project:code-review-agent Review everything we just built
```

## Agent Categories

### Product & Strategy
- **PM Agent** — The orchestrator. Start here when you're unsure what to do next.
- **Research Agent** — Market data, competitor analysis, user research.
- **Innovation Agent** — Future features, emerging tech, creative ideation.
- **Monetization Agent** — Revenue model, pricing, premium features.

### Design & Experience
- **UX Agent** — User flows, friction reduction, interaction patterns.
- **Design Agent** — Visual system, components, spacing, color, typography.
- **Content Agent** — In-app copy, notifications, empty states, brand voice.
- **Accessibility Agent** — WCAG compliance, screen readers, inclusive design.

### Engineering & Quality
- **Code Review Agent** — Code quality, patterns, bug detection.
- **QA Agent** — Test strategy, test writing, coverage analysis.
- **Performance Agent** — Speed, memory, frame rates, bundle size.
- **Security Agent** — Auth, RLS, data protection, vulnerability assessment.

### Infrastructure & Growth
- **Scalability Agent** — Architecture, database, real-time systems.
- **DevOps Agent** — CI/CD, builds, deployment, monitoring.
- **Analytics Agent** — Event tracking, funnels, metrics, dashboards.
- **Marketing Agent** — Growth, ASO, user acquisition, positioning.
